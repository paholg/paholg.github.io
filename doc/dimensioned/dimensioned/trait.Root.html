Trait for implementing general integer roots for types that don't impl Float. Uses Peano numbers to specify the degree.

The syntax is a little bit weird and may be subject to change.

Associated Types

type Output

Example

use dimensioned::si::m;
use dimensioned::{P4, Root};

let x = 2.0*m;
let y = 16.0*m*m*m*m;
assert_eq!(x, P4::root(x*x*x*x));
