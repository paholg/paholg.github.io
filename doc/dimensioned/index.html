cgs

The cgs module provides a unit system for use with Gaussian CGS units.
dimensioned

This module allows dimensioned to be very flexible. It creates the Dim<D, V> type, which is the type that will be used with all dimensionful objects. It then implements as many traits from std as generically as possible.
peano

Peano numbers allow us to do arithmetic at compile time using Rust's type system.

The si module provides a unit system for use with SI units.
